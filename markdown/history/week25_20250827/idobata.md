## まだ終わっていないタスクとその議論

### [PR #463](https://github.com/digitaldemocracy2030/idobata/pull/463)  
**タイトル:** Add real-time updates for question page opinions  
**作成者:** blu3mo  
**作成日:** 2025-08-20T09:40:54Z  

#### どんな変更が議論されているのか？
- 質問ページの「意見」更新をリアルタイムに行うためのWebSocket導入を検討中。  
- 新しい問題／解決策がチャットから送信されると、ページをリロードしなくても自動で意見リストが更新される仕組みを追加。  
- 重複する意見を防止するためにIDチェックを実装し、既存のチャット機能とページ更新機能を分離する設計にする予定です。

#### どんな議論がされている？
- WebSocket をバックエンドとどう連携させるか。  
- ステート管理をどうシンプルに保つか。  
- リアルタイム更新に付随するパフォーマンスへの影響。

#### 参加するには
- リアルタイム更新周りの技術に詳しい方、あるいはフロントエンドのイベント管理に興味がある方は、ぜひレビューやコメントでのフィードバックをお待ちしています！

### [PR #462](https://github.com/digitaldemocracy2030/idobata/pull/462)  
**タイトル:** Add expand/collapse functionality to other opinions section  
**作成者:** Shutaro+Devin  
**作成日:** 2025-08-20T09:25:17Z  

#### どんな変更が議論されているのか？
- 「ほかの人の意見」セクションに、もっと見る／折りたたみボタンを追加してすべての意見を表示できるようにする機能です。  
- これまでは4件までしか表示されなかった意見を、必要に応じてユーザーが展開できるようになります。  
- グラデーションオーバーレイの削除やスクロールバーの制御など、UI/UXの改善が主目的です。

#### どんな議論がされている？
- 大量に意見がある場合のパフォーマンス。  
- モバイルデバイスでの表示や操作性。  
- 展開・折りたたみ時のアニメーションやアクセシビリティ対応。

#### 参加するには
- フロントエンドのUI/UX改善に関心がある方、TypeScriptやリアクティブなUI要素の最適化に知見のある方のレビューが大歓迎です。  
- 実際の表示確認やアクセシビリティに対するフィードバックもお待ちしています。
